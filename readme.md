# aicoder 极简的Blog by NodeJS

aicoder 是基于 node 写的简易的Blog发布系统.[在线demo](http://aicoder.com)

## 系统应用的技术

+ NodeJS `version:5.0`
+ Express 
+ 数据存储使用  MongoDb

## 依赖的NodeJS 包

```plaintext
── body-parser@1.13.3
├── cookie-parser@1.3.5
├── debug@2.2.0
├── express@4.13.4
├── htmlparser2@3.9.1
├── mongoose@4.5.2
├── morgan@1.6.1
├── serve-favicon@2.3.0
├── ueditor@1.1.1
├── xtemplate@4.5.1
└── xtpl@3.2.0
```
